Hello!

This set of tutorials will walk through the Android developer trainings and translate things into Ruby. The two styles of development are actually very different, so it wont be a word-for-word translation. Instead, my goal is to produce the same final programs.

These tutorials will get you started developing in Ruboto, but they aren't really worthwhile unless they also teach you how to use the Android documentation. So throughout these tutorials I will introduce you to the Android docs, and how to translate them to JRuby. When we get to that, I'll assume that you're reading along with me, so please follow the links!
